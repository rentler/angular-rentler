# angular-rentler
Tools to help streamline angular applications. Includes data services and validation.

## Installation

```
bower install angular-rentler --save
```

## Usage

Reference `rentler.core` as a dependency:

```js
angular
  .module('app', ['rentler.core'])
  .run(...);
```
